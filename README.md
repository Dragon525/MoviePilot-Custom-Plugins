# MoviePilot 自定义插件仓库

## 插件列表

### SingleFileFolder - 单文件种子自动创建文件夹

**功能**：
- 监听下载添加事件，记录单文件种子信息
- 定时检查已完成的下载任务
- 对单文件种子：自动创建以电影中文名命名的文件夹，将文件移入
- 同步更新 qBittorrent 种子的保存位置，不影响做种

**使用方法**：
1. 在 MoviePilot 插件设置中添加自定义插件源：
   ```
   https://raw.githubusercontent.com/<你的用户名>/<仓库名>/main/package.v2.json
   ```
2. 在插件市场中搜索 "SingleFileFolder" 并安装
3. 启用插件并配置选项

**插件配置**：
- **启用插件**：开关插件功能
- **强制使用中文名**：如果识别不到中文标题则跳过处理

**注意事项**：
- 仅处理单文件种子（种子内只有一个视频文件）
- 多文件种子（已有文件夹结构）不做处理
- 自动移动视频文件及附属文件（nfo、字幕、海报等）
- 自动更新下载器保存位置，不影响继续做种
